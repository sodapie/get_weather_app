### 取得できる情報
- `実績日`: 天気予報の対象日
- `天気予報発表日`: 実績日の天気予報が発表された日
- `天気予報`: 予報された天気
- `降水確率`: 予報された降水確率
- `最高気温`: 予報された最高気温
- `最低気温`: 予報された最低気温
